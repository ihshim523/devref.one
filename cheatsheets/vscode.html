# VS Code Visual Studio Code quick reference.

General Shortcuts

ShortcutAction
Ctrl+Shift+PCommand Palette
Ctrl+PQuick Open file
Ctrl+Toggle Terminal
Ctrl+BToggle Sidebar
Ctrl+JToggle Panel
Ctrl+WClose tab
Ctrl+Split editor

Editing

Ctrl+D              # select next occurrence
Ctrl+Shift+L        # select all occurrences
Alt+Up/Down         # move line
Shift+Alt+Up/Down   # copy line
Ctrl+Shift+K        # delete line
Ctrl+/              # toggle comment
Ctrl+Shift+\        # indent line
F2                  # rename symbol

Multi-Cursor

Alt+Click           # add cursor
Ctrl+Alt+Up/Down    # add cursor above/below
Ctrl+D              # add selection at next match

Find & Replace

ShortcutAction
Ctrl+FFind in file
Ctrl+HReplace in file
Ctrl+Shift+FFind in files
Ctrl+Shift+HReplace in files
F3 / Shift+F3Next / previous match

Debugging

F5         # start / continue
F10        # step over
F11        # step into
Shift+F11  # step out
Ctrl+F5    # start without debug

Integrated Terminal

Ctrl+`             # toggle terminal
Ctrl+Shift+`       # new terminal
Ctrl+Shift+5      # split terminal
{number} Alt+{num}  # switch terminal
